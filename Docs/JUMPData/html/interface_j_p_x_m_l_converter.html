Detailed Description

JPXMLConverter is a simple XML parser.

It only parse XML documents into a NSDictionary or NSArray. It does not support any fanciness like Namespaces, DTDs, etc.

Repeated elements will be converted to array, as follow: 

 <root>
   <element>
       <data>Element 1</data>
   </element>
   <element>
       <data>Element 2</data>
   </element>
 </root>

Will become an NSDictionary as follow: 

 Dictionary = { root: [ {data: "element1"}, {data: "element2"} ] }

Attributes will be included on the Dictionary, as follow: 

 <root>
    <element attribute="value1">
         Element value.
    </element>
 </root>

Will become an NSDictionary as follow: 

 Dictionary = { root: {attribute: "value1", element:"Element value."} }
