JUMP Data Overview

JUMP Data Module is an collection of classes to process and convert data of many types. Classes to convert JSON objects, Objective-C data types and Math helpers are some examples.

Navigate trough the Class List to known more about this classes.

Note:
Import the JUMPData.h header to your project to convenient acces all classes of this module.
