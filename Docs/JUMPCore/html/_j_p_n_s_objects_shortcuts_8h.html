#define NSReleaseSafely (   __POINTER)    { [__POINTER release]; __POINTER = nil; }

Shortcut to safely release an object.

This macro function send an release message to the object and then nullify him. This is a safe technique because when we release an Objective-C object and then call him you have an exception.
